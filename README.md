# Automated_Mixer

Intelligent Foam Controlled Mixing System is an automated industrial mixing solution designed to control foam formation during the mixing of herbal shampoos and cosmetic liquids. The system uses an ESP32 microcontroller, ultrasonic sensor, and motor driver to monitor foam levels in real time and automatically adjust the agitator motor speed. This helps reduce overflow, improve mixing efficiency, minimize product wastage, and ensure uniform product quality through smart monitoring and control.
